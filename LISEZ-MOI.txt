CMFG-UniversShop — Site marketplace responsive
================================================

Contenu de l'archive :
- index.html        : le site complet (HTML + CSS + JS dans un seul fichier)
- images/           : dossier vide à remplir avec vos photos produits
- LISEZ-MOI.txt     : ce fichier

UTILISATION
-----------
1. Ouvrez index.html dans n'importe quel navigateur pour tester en local.
2. Pour publier en ligne, déposez le dossier complet (index.html + images/)
   chez votre hébergeur (OVH, Netlify, Hostinger, Infomaniak, etc.).
